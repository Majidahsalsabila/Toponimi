# Toponimi
Kajian Toponimi Kampung Betawi: Penelitian ini  mengeksplorasi hubungan nama “Joglo” dengan arsitektur tradisional Jawa-Betawi dan perubahan persepsi akibat urbanisasi dalam konteks toponimi Kampung Joglo dengan menggunakan pendekatan semantik Teori Segitiga Semantik Ogden &amp; Richards (1923) dan Cruse (2000) serta pragmatik Chaer (2008).
